# G, gadgetdir

> `gogo G path/to/gadgets`

Set the path of the folder containing the config files.

**_NOTE: You need to specify the full path from the root directory_**

<Tabs>
  <TabItem value="G" label="G" default>

    gogo G /home/me/.gogo/gadgets
  </TabItem>

  <TabItem value="gadgetdir" label="gadgetdir">

    gogo gadgetdir C:\Users\Me\gogo\gadgets
  </TabItem>
</Tabs>
