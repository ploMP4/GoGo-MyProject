# T, templatedir

> `gogo T path/to/templates`

Set the path of the folder containing the templates.

**_NOTE: You need to specify the full path from the root directory_**

<Tabs>
  <TabItem value="T" label="T" default>

    gogo T /home/me/.gogo/templates
  </TabItem>

  <TabItem value="templatedir" label="templatedir">

    gogo templatedir C:\Users\Me\gogo\templates
  </TabItem>
</Tabs>
